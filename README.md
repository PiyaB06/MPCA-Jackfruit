# MPCA-Jackfruit ~ Smart Medicine Reminder Box 

## Abstract

This device reminds you to have your medications on time. It allows you to set multiple alarms a day, at any time as per your convenience. It uses an Arduino Uno as the brain, and modules like DS3231 Precision RTC to keep accurate time. It uses an LCD display to allow the user to see and set the time.  It includes buttons for toggling, and a buzzer as a reminder. 


## Hardware Components Used:
- Breadboard
- LCD
- RTC Module
- Male to Male Jumper Wires
- Male to Female Jumper Wires
- Push Buttons (x2)
- Buzzer
